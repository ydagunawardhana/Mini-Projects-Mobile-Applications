# 📱 Mini-Projects-Mobile-Applications

Welcome to my collection of mobile application mini-projects! This repository contains a set of simple, yet functional applications created to explore UI design, interactivity, and CRUD operations.

## 🚀 Projects Included

### 1. 👤 Profile Details App (`mini_project01_profile_page`)
A modern and minimalist profile screen layout built with Flutter.
* **Features:**
  * Displays user avatar with an active status indicator.
  * Shows user details including Name and Email.
  * Tracks user points or rewards.

### 2. 🏏 Mini Cricket Score App (`mini_project02_cricket`)
A fun, interactive tap game for quick entertainment.
* **Features:**
  * Tap the "Bat" button to play a ball.
  * Tracks total runs scored and balls faced.
  * Dynamic run status updates (e.g., "No Runs", boundaries).

### 3. ✅ Task Management / CRUD App (`crud/crud` & `lib`)
A clean task management application demonstrating Create, Read, Update, and Delete (CRUD) operations.
* **Features:**
  * Add new tasks effortlessly.
  * Mark tasks as completed.
  * Delete tasks using the bin icon.
  * Search functionality to easily find specific to-dos.
